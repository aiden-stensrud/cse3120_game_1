# CSE 3120 Project # 1
By Prottasha Deb and Aiden Stensrud<br>
Our game is "Hangman." A user inputs a string of letters, which must then be "guessed" by a player by inputting letters into the program. The player has a limited number of guesses.
### Compilation Instructions
* To assemble the program, you must have a Windows machine with Visual Studio 2015 installed.
* Navigate into the ```cse game 1/src folder``` and run ```assemble.bat```
* The program is now assembled and can be run with ```hangman.exe```
